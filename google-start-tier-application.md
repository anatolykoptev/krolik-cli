### Product description (short)
```
Multi-Tier AI Orchestration CLI tool for TypeScript developers. Automatically routes AI tasks to optimal model tiers, reducing costs by 90% through intelligent model selection.
```

### Product description (detailed - if asked)
```
Krolik Felix is an open-source CLI tool that helps developers optimize AI API costs through intelligent multi-tier model routing.

The system analyzes task complexity and automatically selects the most cost-effective AI model from four tiers (Free, Cheap, Mid, Premium). Simple tasks like code formatting use free models, while complex tasks like architecture design use premium models.

Key features:
- Automatic model selection based on task complexity
- Cascade fallback for reliability
- SQLite-based learning from historical success patterns
- Native TypeScript integration
- 90% cost reduction compared to always using premium models

Target users: TypeScript developers, AI application builders, startups optimizing AI costs.
```
